# gitbook-plugin-signature

## book.json

```json
"plugins": ["gitbook-plugin-signature@git+https://github.com/popodidi/gitbook-plugin-signature.git"]
```

## Usage

### created/modified

```
{{ 'Hao'|created}}
{{ 'Hao'|lastModified}}
```

### Last Modified Timestamp

```
**lastModifiedTimestamp**
```

### copyright

```
{{ 'Hao'|copyright }}
```
